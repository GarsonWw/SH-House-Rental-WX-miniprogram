# 云开发接入指南

按以下步骤操作，5-10分钟完成接入。

---

## 第一步：开通云开发环境

1. 打开**微信开发者工具**，点击顶部工具栏 **"云开发"** 按钮
2. 点击 **"开通"**，填写环境名称（如 `rentapp`），选择**免费套餐**
3. 等待创建完成（约1分钟），复制生成的**环境 ID**（格式类似 `rentapp-xxxxxxxx`）

---

## 第二步：填入环境 ID

打开 `miniprogram/app.js`，将第3行的 `YOUR_ENV_ID` 替换为你的环境 ID：

```js
const CLOUD_ENV = 'rentapp-xxxxxxxx'   // ← 填你自己的
```

---

## 第三步：创建数据库集合并设置权限

1. 在云开发控制台点击 **"数据库"**
2. 点击 **"+"** 新建集合，名称填 `houses`
3. 房源写操作现在统一通过 `houseService` 云函数完成，请继续按照根目录 `CLOUD_SECURITY_SETUP.md` 创建管理员名单并设置安全规则。
4. `houses` 和 `neighborhoods` 应设置为所有用户可读、客户端不可写。

---

## 第四步：初始化示例数据（仅需一次）

1. 在微信开发者工具左侧找到 `cloudfunctions/seedHouses` 文件夹
2. **右键** → **"上传并部署：云端安装依赖"**，等待部署完成
3. 部署后，右键同一文件夹 → **"云端测试"**，点击 **"调用"**
4. 看到返回 `"初始化完成：成功 6 条"` 即可

> ⚠️ 只需运行一次！再次调用会自动检测到已有数据并跳过。

---

## 第五步：配置合法域名（发布正式版时需要）

开发阶段在微信开发者工具中勾选 **"不校验合法域名"** 即可。

正式上线前，在**微信公众平台** → **开发** → **开发管理** → **服务器域名**中添加：
- `https://api.weixin.qq.com`（微信云开发已自动配置，通常无需手动添加）

---

---

## 完成！

配置完成后重新编译，房源数据将存储在云端。客户端会在页面重新显示时刷新状态，并在网络异常时明确提示正在展示缓存。

**房东发布的房源** → 所有用户刷新后可见，详情页停留时会定期同步状态
**删除/修改** → 仅 `admins` 管理员名单中的账号可操作
**收藏夹** → 仍存在手机本地，每个用户独立
