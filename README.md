# 🏠 租房找房 · 微信小程序

> 一款专注于小区级别房源共享的微信小程序，支持直接获取房东微信号/电话，零中介费直联。

---

## 📱 功能模块

### 1. 首页（找房）
- 🔍 全局搜索：按小区名、区域、房型搜索
- 🏘️ 热门小区横向卡片，点击直达该小区全部房源
- 🔥 热门推荐 / 最新发布 切换展示
- Banner 轮播

### 2. 小区页（按小区找）
- 区域筛选芯片（深圳罗湖区等）
- 小区网格卡片展示（含小区名、区域、在租套数）
- 点击小区 → 展示该小区所有房源
- 房源卡片直接显示「拨打电话」「复制微信」按钮

### 3. 房源详情页（核心）
- 完整房源信息（房型、面积、楼层、朝向、装修）
- **房东信息卡片**：
  - 📞 电话联系 → 弹窗确认 → 直接拨打
  - 💬 微信联系 → 弹窗展示微信号 → 一键复制
  - 步骤引导：复制 → 打开微信 → 搜索添加
- 底部固定联系栏（始终可见）

### 4. 发布房源页
- 3步向导表单：
  - Step 1: 房源信息（小区、价格、房型、图片、标签）
  - Step 2: 房东联系方式（姓名、电话、微信号）
  - Step 3: 预览确认后发布
- 本地存储，发布后立即可见

### 5. 我的页面
- 发布数量、可租数量、总浏览量统计
- 我发布的房源列表
- 一键标记「已租/重新出租」
- 删除房源

## 🚀 快速开始

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，点击「导入项目」
3. 选择本项目根目录 `d:\ZFXCX`
4. 填入你的小程序 AppID（可使用测试号）
5. 点击编译即可预览

> ⚠️ 需要在 `project.config.json` 中将 `appid` 替换为你自己的小程序AppID

## 💡 数据存储

- 当前版本使用**微信本地存储**（`wx.getStorageSync / setStorageSync`）保存房源数据
- 预置了 6 条示例房源数据
- 生产环境建议接入云开发（微信云数据库）或自建后端API
